# Create a Project

## **Introduction**
This guide will help you create a **Project** in the programme.

## **Prerequisites**
1.	**User ID** and **Password** to access APXGIS programme.
2.	Assigned **Admin Rol**e to the given user ID.

**Version**: 00
**Date**: 22-09-2023
