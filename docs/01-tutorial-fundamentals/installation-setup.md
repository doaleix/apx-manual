# Installation Setup

This guide will help you **Setup your APX Installation**.

**Prerequisites**
1.	**User ID** and **Password** to access APXGIS programme.
2.	Assigned **Admin Role** to the given user ID.


------------

**Version**: 5.5
**Date**: December 2025

------------

## **Step by Step**

Click "**Administration**" dropdown menu.

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-02.png)

Click "**Installation setup**" menu option and the setup window will come up.

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-03.png)

### Fields

"**Code**" this number is given by the programme for the installation.

![](/img/MNG-INS-CNF-01/setup-page.png)

1\. Click "**Name**", to set up the installation name, this name will be displayed on the top right corner, next to the company logo. For this example it was typed "**Company name**".

2\. Click "**Choose**" button to select the company logo for the installation.

The file explorer window will open. Search for the logo you want to use and click "**Open**" button.

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-06.png)

3\. Click "**Lang**" dropdown (Language) to select the installation language and click one of options. For this example, **English** has been selected.

4\. To go to the alternative projection screen click the "**Edit**" button

In the alternative projection screen, users can search by the projection name or the SRID code. 

![](/img/MNG-INS-CNF-01/alt-proj-02.png)

Once a projection is selected, the form displays the extended information of the projection in read-only input fields and a map area that displays the effective area of the projection.

![](/img/MNG-INS-CNF-01/alt-proj-03.png)

By selecting an alternative projection, the corresponding coordinates will be displayed in element forms of type node (points in the map)

![](/img/MNG-INS-CNF-01/alt-proj-04.png)

And in the map pop up that comes up with a left mouse click

![](/img/MNG-INS-CNF-01/alt-proj-05.png)

## Advanced settings

### Params

5\. Click "**Params**" button to display all the parameters.

"**Params**"  (parameters) window will be displayed. 

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-09.png)

These parameters are set according to the needs of the particular installation or client.

Click "**Close**" button to close "**Params**" window.

### Map Center Setup

6\. Click "**Map Center Setup**" button.

The "**Map Center Setup**" window will be shown. This screen purpose is to set a default initial location and zoom when the application starts up.

To set it, first go to the desired location  by scrolling through the map and select the appropiate zoom.

Then click the **get current values** button.

![](/img/MNG-INS-CNF-01/alt-proj-07.png)

After the values have been updated remember to hit the **Apply** button to save the changes.  

### Themes

7\. Click "**Themes**" button. Themes are created to define the style in which you want to display an element. They are temporary or informative states of the elements.

Click "**Add**" button to add a new Theme.

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-16.png)

Click "**Roles**" dropdown to select the roles that will have access to the Theme to be created.

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-17.png)

Click the **selection box** of the role (s). For this example "**admin**" role.

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-18.png)

Click "**Name**" text field (1). Type the name of the "**Theme**" to be created (2), for this example "**Theme-01**" and click "**Apply**" button (3).

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-19.png)

Click "Close" button.

![](/img/MNG-INS-CNF-01/MNG-INS-CNF-01-STP-20.png)

### API Token

8\. Click the API Token button to manage API tokens in your installation.

![](/img/MNG-INS-CNF-01/api-token01.png)

To know more techical details about APX's API Click the Api Documentation Hyperlink or <u>[here](https://app.apx-gis.net/static/documentation/api/)</u>

A new token can be created by clicking the **+Add** button.

![](/img/MNG-INS-CNF-01/api-token02.png)

The Api Tokens management screen will open with a blank form to fill up the following fields:

* A) This is the only mandatory field, free text field to input the name of the Token.

* B) The token number is a read-only field and will be autogenerated when creating the token.

* C) Pick a date if you want the token to have a definite validity. It can be modified later on. Leave the field blank if you don't want the token to expire ever.

* D) The default number of Maximum requests per minute is 1000. Talk to our support team if you want this to have a different value. It can't be changed through the user interface.

* E) Free text to leave any relavant notes regarding this token.

* F) Set up the privileges of this token regarding the API endpoint described in the Api Documentation.

When all field are filled as required click the **Apply** button to create the token.

When a token is created an associated User is created to represent the token in the <u>[User management](../02-Advanced-Admin/01-tutorial-user-management/01-tutorial-users/01-user-screen.md)</u>

An already created token can be modified by clicking the row in the token list:

![](/img/MNG-INS-CNF-01/api-token03.png)

When updating an existing token, a new token number can be regenerated by clicking the **Regenerate** button.

A new button will be displayed under the Comments field to acces the associated User to the API token.

### Clean Caches

This is an action button to erased all data previously cached. It is recommended to avoid using it unless it is necessary since it can potentially reduce system performance.
